# Habit Tracker

A simple menu-driven Habit Tracker built using Python.

## Features
- Add new habits
- Display all habits
- Mark habits as completed
- Delete habits
- Uses a Python dictionary to store habit status
- Menu-driven interface

## Technologies Used
- Python 3

## Concepts Practiced
- Functions
- Dictionaries
- Loops
- Conditional Statements
- User Input

## How to Run

1. Clone the repository.
2. Open the project in VS Code.
3. Run the Python file.

```bash
python habit_tracker.py
